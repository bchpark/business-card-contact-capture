# 명함 연락처 변환기

명함 사진을 브라우저에서 OCR로 읽고, Google 연락처에서 가져올 수 있는 CSV 또는 vCard 파일로 내보내는 웹앱입니다.

## 실행

```bash
npm start
```

PowerShell 실행 정책 때문에 `npm`이 막히면 아래 명령을 사용합니다.

```bash
node server.js
```

브라우저에서 `http://localhost:3000`을 엽니다.
3000번 포트가 사용 중이면 서버가 자동으로 다음 포트를 사용합니다.

## 사용 흐름

1. 명함 사진을 선택하거나 휴대폰 카메라로 촬영합니다.
2. `글자 인식`을 누릅니다.
3. 자동 추출된 이름, 회사, 전화번호, 이메일 등을 확인하고 수정합니다.
4. `Google CSV`를 내려받아 Google 연락처의 가져오기 메뉴에서 업로드합니다.

## 한글 인식 팁

- 기본값인 `한글 강화`는 명함을 3배 확대하고 대비/샤픈 보정을 적용합니다.
- 흐릿하거나 작은 글자는 `작은 글자 선명`을 사용해 보세요.
- 로고나 배경색 때문에 글자가 뭉개지면 `원본에 가깝게`로 바꾸는 편이 낫습니다.
- 사진은 명함을 화면 가득 채우고, 그림자 없이 밝게 찍을수록 인식률이 좋아집니다.
- 전체 인식 후 회사 이름은 파란색, 사람 이름은 초록색, 연락처는 빨간색 사각형으로 표시됩니다.
- 각 사각형 위치가 틀리면 드래그하거나 모서리를 잡아 조절합니다. 손을 떼면 이동한 사각형 하나만 자동으로 다시 인식합니다.
- 필요하면 `모든 영역 다시 인식`으로 회사/사람/연락처를 한 번에 다시 읽을 수 있습니다.

## 참고

OCR은 브라우저에서 Tesseract.js CDN을 불러와 실행합니다. 처음 사용할 때 인터넷 연결이 필요합니다.
